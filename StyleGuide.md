### Guidelines for emoji in commit messages

🎓 - Localisation changes

🌸 - UI improvement

👽️ = Bug fix

✨ = New user-facing feature

🧱 = New codebase feature/feature otherwise not reaching all users

⚙️ - Developer tooling

⚡️ = CI pipelines

📘 = Documentation, wikis

🏷️ = Releases

💚 = Community content & Automated commits
